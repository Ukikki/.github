## 🐬 **DolAi**

> <i>"방금 무슨 얘기했더라?"</i><br>
> <i>"누가 이거 하기로 했지?"</i><br>
> <i>"회의록은 언제 정리돼요?"</i>

DolAi는 이러한 **회의 정보의 손실**을 막기 위해,  
**회의별 독립적인 Graph RAG를 실시간으로 동적 생성하는 AI 어시스턴트**입니다.
